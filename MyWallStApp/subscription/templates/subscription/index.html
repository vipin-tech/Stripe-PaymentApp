MyWallSt Backend Developer Assignment

We would like you to prepare a backend application that handles Stripe payments, written in Python + Django and Django REST framework. The purpose of this application is to expose an API through which Stripe payments can be initiated, and where subscription statuses are processed and stored. With this assignment you can demonstrate your experience with Django and API design and integration with third party APIs, and show off your (technical) creativity.
